What is Morse Code?
Morse code is a method of encoding text characters as sequences of dots (short signals) and dashes (long signals). It was created in the early 1830s and 1840s by Samuel Morse and Alfred Vail for use in telegraphy. Each letter of the alphabet, each digit, and certain punctuation marks are represented by a unique sequence of dots and dashes.
